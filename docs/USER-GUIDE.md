## 3. 오디언스 만들기 — 기본 필터

필터는 **여러 개를 동시에 적용**할 수 있고, 조건을 바꾸면 상단 요약 카드가 즉시 갱신됩니다.

| 필터 | 설명 |
|---|---|
| **성별** | 남성 / 여성 / 알수없음 |
| **연령** | 10대 ~ 60대+ / 미상 (복수 선택 가능) |
| **지역** | 시도(17) → 시군구 → 읍면동 단계 선택 |
| **업종** | 대분류(13) → 중분류 → 소분류 |
| **쇼핑** | 쇼핑 카테고리(14) |
| **금액** | 금액 구간 또는 상위/하위 10·20·30% |
| **매체** | 매체·카드사(10) |
| **통신 / 단말** | 통신사, 단말 종류 |
| **소비** | 소비 대카테고리(13) |
| **결제** | 승인횟수 / 승인금액 + 기간(최근 90일 등) + 조건(이상·이하·구간) |
| **ADID** | 보유 광고식별자 파일 업로드 → 매칭 |
| **기간** | 1개월 / 3개월 / 6개월 / 1년 / 직접 선택 |

### ADID 파일 업로드
1. `+ ADID 파일 업로드` 클릭
2. 광고식별자 목록 파일 선택
3. 매칭 결과 표시: `📤 12,345건 매칭 / 20,000건 업로드 (61.7%)`

> 업로드한 목록과 DMP 데이터가 겹치는 대상만 매칭됩니다.

### 선택한 조건 확인
적용된 필터는 상단에 **태그**로 표시됩니다. 태그의 `✕` 를 눌러 개별 해제할 수 있습니다.

### 📊 매체 성과 탭 (자세히)

#### 상단 — 요약
| 지표 | 의미 |
|---|---|
| 총 노출 | 선택 기간 광고 노출 수 |
| 총 전환 | 실제 전환 발생 수 |
| 총 광고비 | 집행 광고비 |
| 활성 매체 | 노출이 발생한 매체 수 |

기간은 **7일 / 30일 / 90일** 중 선택합니다.

#### 시각화 전환
화면 우측 상단의 **시각화** 버튼으로 표현 방식을 바꿀 수 있습니다.

| 옵션 | 특징 | 추천 용도 |
|---|---|---|
| **현재** | 간단한 막대 | 빠른 확인 |
| **정제 2D** | 효율 진단 버블 · 매체 랭킹 · 전환 점유 · 추이 | **일상 업무 권장** |
| **🧊 3D** | 날짜×매체×노출 입체 지형 | 패턴 탐색 |
| **✦ BizViz** | 코스믹 스타일 | 발표·보고용 |

> **정제 2D의 「매체 효율 진단」** 읽는 법
> 가로축 = 광고비, 세로축 = 전환율, 원 크기 = 노출량
> → **왼쪽 위에 있는 큰 원** = 적은 비용으로 높은 전환을 낸 매체

#### 하단 좌측 — 매체 순위
- 매체별 노출·클릭·전환·광고비 (최대 50행)
- **행을 클릭**하면 해당 매체의 일별 추이만 표시
- 🔁 배지 = 소재 반응 데이터를 보유한 매체

#### 하단 우측 — 🔁 오디언스 반응 소재 (폐루프)
1. 드롭다운에서 **오디언스 선택**
2. 그 오디언스가 실제로 **전환한 광고 소재 TOP** 표시
3. 좌측에서 매체를 선택하면 해당 매체로 자동 필터

> **결과가 비어 있어도 정상인 경우가 있습니다.**
> 적립형 매체(출석체크·적립 등)는 광고 소재명 없이 전환이 기록되어 소재별 분석 대상이 아닙니다.
> 소재 반응은 🔁 배지가 있는 매체에서 확인해 주세요.

## 5. AI 탐색 — 오디언스 자동 발굴

조건을 직접 지정하는 대신, **AI가 목적에 맞는 오디언스를 찾아 제안**합니다.

```
탐색 요청  →  AI 후보 생성  →  미리보기(규모 확인)  →  승인 / 반려
```

1. **🧪 AI 탐색** 탭 또는 `AI 오디언스 탐색` 버튼
2. 찾고자 하는 대상을 입력
3. AI가 후보 오디언스 생성
4. **미리보기**로 규모·구성 확인
5. **승인**하면 전송 가능한 오디언스로 등록 / **반려** 시 폐기

> 승인된 오디언스는 전송 화면의 소스 목록에 나타납니다.

## 6. 타겟 전송

### 전송 절차
1. 조건 설정 또는 오디언스 선택
2. **🚀 런컴 타겟 전송** 클릭
3. 팝업에서 확인
   - **소스**: 어떤 오디언스인지
   - **대상 규모**: 몇 명인지
4. 전송 실행

### ⚠️ 전송 전 확인
- 전송은 **실제 운영 반영**입니다. 실행 후 취소할 수 없습니다.
- 대상 규모와 소스를 **반드시 확인** 후 실행해 주세요.
- 전송 결과는 **📋 전송 이력** 탭에서 확인합니다.

### 데이터 취급
- 오디언스 데이터는 **런컴 지면 전송 목적으로만** 사용됩니다.
- 제3자 플랫폼으로의 반출은 이루어지지 않습니다.

## 7. 성과 확인

| 확인할 것 | 어디서 |
|---|---|
| 전송한 타겟 목록·상태 | 📋 전송 이력 |
| 매체별 노출·클릭·전환·광고비 | 📊 매체 성과 |
| 일별 추이 | 📊 매체 성과 → 일별 노출·전환 추이 |
| 어떤 소재가 반응을 얻었나 | 📊 매체 성과 → 🔁 오디언스 반응 소재 |
| 매체 효율 (비용 대비 전환) | 📊 매체 성과 → 정제 2D → 매체 효율 진단 |

**권장 운영 사이클**
```
오디언스 생성 → 전송 → (집행) → 매체 성과 확인 → 반응 소재 파악 → 다음 오디언스 개선
```

## 8. 자주 묻는 질문

**Q. 조회가 느립니다.**
대용량 데이터를 처음 조회할 때는 최대 1~2분이 걸릴 수 있습니다. 이후 동일 조건은 수 초 내로 응답합니다. 상단 **응답 속도** 표시로 현재 상태를 확인할 수 있습니다.

**Q. 폐루프(반응 소재) 결과가 비어 있습니다.**
정상일 수 있습니다. 적립형 매체는 소재명 없이 전환이 기록됩니다. 🔁 배지가 있는 매체를 선택해 주세요.

**Q. 필터를 걸었는데 대상이 0명입니다.**
조건이 과도하게 좁을 수 있습니다. 태그를 하나씩 해제하며 규모를 확인해 보세요.

**Q. 전송을 취소하고 싶습니다.**
전송은 실행 즉시 반영되어 취소할 수 없습니다. BizSpring 담당자에게 문의해 주세요.

**Q. 매체 성과의 숫자가 다른 리포트와 다릅니다.**
집계 기준일·기간·매체 범위가 다를 수 있습니다. 기간 선택(7/30/90일)을 먼저 확인해 주세요.

**Q. 데이터는 언제 갱신되나요?**
일 단위 배치로 갱신됩니다. 당일 데이터는 익일 반영됩니다.

## 9. 용어 사전

| 용어 | 의미 |
|---|---|
| **오디언스** | 특정 조건을 만족하는 이용자 집합 |
| **세그먼트** | 오디언스를 나누는 기준 (성별·연령·업종 등) |
| **큐브** | 빠른 조회를 위해 미리 집계해 둔 데이터 |
| **ADID** | 광고식별자 — 이용자 매칭에 사용 |
| **노출 (Impression)** | 광고가 보여진 횟수 |
| **클릭** | 광고를 클릭한 횟수 |
| **전환 (Conversion)** | 광고 후 실제 행동(구매·참여 등)이 발생한 건수 |
| **전환율** | 전환 ÷ 노출 |
| **CTR** | 클릭 ÷ 노출 |
| **폐루프 (Closed-loop)** | 전송한 오디언스가 실제 어떤 소재에 반응했는지 되돌아보는 분석 |
| **적립형 매체** | 출석체크·포인트 적립 방식 매체 (소재명 없이 전환 기록) |
| **소재형 매체** | 개별 광고 소재로 전환이 기록되는 매체 (🔁 배지) |
| **리프트 (Lift)** | 기준 대비 성과 배수. 1.8× = 기준의 1.8배 |
